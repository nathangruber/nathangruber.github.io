Blake's Board Shop is a faux e-commerce website I built to show my proficiency using LAMPP stack. LAMPP stack uses PHP, HTML5, CSS3, mySQL, phpMyAmin, and I also used little bit of Javascript. CRUD(create, read, update, and delete) interface was the foundation of the website. Click on any image in the slideshow to take you to a specfic page in the site. It is fully functional, and feel free to register an account so you can see the functionality. Thank you for visiting, and if you have any questions please contact me.
